## Development

### `npm install && npm start`

## Create image

### `docker build -t izicap .`

## Run

### `docker run -d -p 80:80 izicap`

## Environment variables (in production)

### `docker run -d -p 80:80 -e REACT_CLIENT_ID=<> -e REACT_CLIENT_SECRET=<> izicap`
